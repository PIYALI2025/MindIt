MindIt | Subscription & Bill Tracker
MindIt is a smart backend-driven dashboard designed to help users track their recurring subscriptions, monitor monthly burn rates, and never miss a payment deadline.
Features
Performance Dashboard: Real-time calculation of monthly expenditures and active trials.
Automated Urgency Status: Categorizes bills by "Due Today," "Upcoming," or "Overview" based on the remaining days.
Asynchronous Backend: Built with FastAPI and Motor (MongoDB) for high-performance data handling.
Unique ID Generation: Automatic 8-character UUID generation for every subscription.
Cross-Origin Support: Fully configured CORS middleware for seamless Frontend-Backend communication.

 Tech Stack
Language: Python 3.10+
Framework: FastAPI
Database: MongoDB (NoSQL)
Library: Motor (Async MongoDB Driver)
Frontend: HTML5, CSS3, JavaScript (Fetch API)
